# xunfei voice 
> This plugin is intended to show a picture from an URL into a Photo Viewer with zoom features.

## How to Install

Cordova:
>cordova plugin add 


### Android
> Out of the box

### iOS
> Out of the box


### API

#### Show an image


Optionally you can pass third parameter which can be used to hide the share button [ONLY FOR ANDROID]

##### Usage

```
voice.startVoice();
```
